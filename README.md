# 日本の4G band周波数対応表
jp-4g-band-table

## これは何

スマートフォンなどが対応している4G(LTE)周波数と日本の通信事業者が提供している周波数を照合し、どの事業者の通信に対応しているのかを表示します。

## 使用方法

1. ダウンロードして開くか、[GitHubのURL](https://zaki-system.github.io/jp-4g-band-table/)を開く
2. 入力boxに対応bandを","または"/"区切りで入力し、[表示]を押す
